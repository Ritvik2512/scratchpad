# House Price Prediction using Machine Learning

This project predicts housing prices using the California Housing dataset.

## 🚀 Overview

The goal of this project is to:
- Apply regression models
- Compare performance of different algorithms
- Understand overfitting and model tuning

## 📊 Models Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## 📈 Results

| Model               | MSE  |
|--------------------|------|
| Linear Regression  | ~0.43 |
| Decision Tree      | ~0.49 |
| Random Forest      | ~0.26 |

Random Forest performed the best due to its ability to capture non-linear relationships and reduce overfitting.

## 🧠 Key Learnings

- Linear models struggle with non-linear data
- Decision Trees can overfit without tuning
- Ensemble models like Random Forest improve performance
